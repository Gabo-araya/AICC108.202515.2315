¿Qué es el Phishing?

Definición y Ejemplos

El phishing es una técnica de ingeniería social utilizada por ciberdelincuentes para obtener información confidencial como contraseñas, números de tarjetas de crédito y otros datos personales.

Ejemplos Comunes:
  • Email bancario falso: "Su cuenta será suspendida. Haga clic aquí para verificar"
  • Sitio de PayPal clonado: Página idéntica que roba credenciales
  • SMS de banco: "Confirme su tarjeta en este enlace urgente"
  • Red social falsa: Login de Facebook/Instagram clonado
Importante: Una de las técnicas más comunes del phishing implica la clonación de sitios web legítimos para engañar a las víctimas.
1
Clonación del Sitio

El atacante crea una copia exacta de un sitio web legítimo

2
Distribución

Envía enlaces maliciosos por email o redes sociales

3
Captura de Datos

Recolecta la información ingresada por las víctimas

Clonación del sitio web con wget

Proceso de clonación del sitio web de "Ficticia Inc." usando wget

Sitio Clonado Sitio Original
Comando wget para clonación completa
wget --recursive --no-clobber --page-requisites --html-extension \ --convert-links --restrict-file-names=windows --domains xqazprog.pythonanywhere.com \ --no-parent https://xqazprog.pythonanywhere.com/
Parámetros explicados:
  • --recursive: Descarga recursivamente
  • --page-requisites: Descarga CSS, JS, imágenes
  • --convert-links: Convierte enlaces para uso local
Características:
  • ✓ Mantiene estructura de carpetas
  • ✓ Enlaces funcionan localmente
  • ✓ Respeta robots.txt

Importancia de Reconocer Páginas Maliciosas

Aprende a protegerte contra ataques de phishing y sitios web maliciosos

¿Por qué es importante?

  • Proteger información personal y financiera
  • Evitar el robo de identidad
  • Prevenir pérdidas económicas
  • Mantener la seguridad de las organizaciones

Estadísticas Alarmantes

  • 91% de los ciberataques comienzan con phishing
  • 1 de cada 99 emails es un intento de phishing
  • 76% de las organizaciones fueron víctimas de phishing
  • $12 millones es la pérdida promedio por ataque

Consejos para Evitar Caer en Phishing

Verificación de URLs
  • Revisa cuidadosamente la URL antes de hacer clic
  • Busca el candado de seguridad (HTTPS)
  • Desconfía de URLs acortadas o sospechosas
  • Verifica el dominio oficial de la empresa
Análisis de Emails
  • Verifica la dirección del remitente
  • Busca errores ortográficos y gramaticales
  • Desconfía de la urgencia excesiva
  • No descargues archivos adjuntos sospechosos
Protección Personal
  • Usa autenticación de dos factores (2FA)
  • Mantén actualizado tu navegador y antivirus
  • Nunca ingreses datos sensibles en sitios no verificados
  • Educa a tu familia y colegas sobre phishing
Señales de Alerta
  • Solicitudes urgentes de información personal
  • Ofertas demasiado buenas para ser verdad
  • Amenazas de suspensión de cuentas
  • Formularios de login en emails
